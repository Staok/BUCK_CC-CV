程序框架：

main()
{
    init();
    while(1)
	{
	    屏显信息();//有变化时才更新;if(故障 != 0){显示故障信息();}
	}
}

定时器中断10ms()
{
    采集所有数据并处理();故障诊断();判断当前模式并切换相应mode():所有任务();
}

所有任务()
{
    是否进入mode_1→执行...
    是否进入mode_2→执行...
    ...
}

故障诊断()
{
    如果 故障1发生 → return 1;
    如果 故障2发生 → return 2;
    ...
    else return 0;//0表示无故障
}

最高优先级定时器中断1ms()
{
    计算PID任务();
}
